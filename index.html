🤖 Telegram Bot Setup

Step 1 — Create your bot

Message @BotFather on Telegram → /newbot → get your Bot Token

Step 2 — Get your Chat ID

Message @userinfobot on Telegram to get your group or personal Chat ID

Step 3 — Deploy the bot server

Upload bot.js to Railway (free tier). Set env vars:
BOT_TOKEN, SUPABASE_URL, SUPABASE_KEY, ALLOWED_CHAT_ID

Step 4 — Set webhook

Visit: https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://<your-railway-url>/webhook
